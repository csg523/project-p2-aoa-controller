Module Responsibilities
=======================

hal_input.c/h
  Input:  UART messages ($AOA, $FLIGHT_PARAMS, $FLIGHT_MODE)
  Output: Circular buffers, EventGroup signal
  Sync:   Mutex protection on buffer writes

validator.c/h
  Input:  Circular buffers (newest values)
  Output: Median AoA, VALID/INVALID sensor flags
  Math:   Median calculation, outlier detection (±5°)

estimator.c/h
  Input:  VALID sensor values from validator
  Output: Final stabilized AoA estimate
  Math:   Weighted fusion (1/e weights), 1D Kalman filter

fsm.c/h
  Input:  Calculated AoA from estimator
  Output: FSM state, LED control signals
  Logic:  Thresholds table lookup, state machine transitions

logger.c/h
  Input:  FSM output, sensor data, calculated AoA
  Output: UART log message, LED GPIO control
  Format: IDX TS MODE S1 S2 S3 AIRSPEED AOA STATUS


Execution Flow
==============

[Hardware Timer: 20ms Period]
           ↓
[EventGroup: BIT_RUN_CYCLE]
           ↓
[Control Task Wakes]
           ├─→ [VALIDATOR] Read buffers, calculate median
           ├─→ [ESTIMATOR] Weighted fusion + Kalman filter
           ├─→ [FSM] State machine transitions
           ├─→ [LOGGER] Output LED & UART
           └─→ [Sleep] Wait for next cycle

[Async: UART Reception]
           ↓
[HAL_INPUT Task] Parse messages, update circular buffers


Data Protection
===============

Usage:

Key Algorithms
==============

Median(a, b, c):              O(1) comparison-based
Outlier Detection:            |sensor - median| > 5.0°
Weighted Fusion:              W=1/e normalized weights
Kalman Filter (1D):           Predict → Update → Store

Thresholds:
  NORMAL:      AoA < (High - 2.0°)
  CAUTION:     AoA ≥ (High - 2.0°)
  PROTECTION:  AoA ≥ (High - 0.5°)
  OVERRIDE:    AoA ≥ High


Configuration
==============

Control Cycle:     20ms (50Hz)
Sensor Buffer:     5 values per sensor
Outlier Margin:    5.0°
Kalman Q:          0.01 (process noise)
Kalman R:          0.5 (measurement noise)
LED Caution:       500ms blink
LED Protection:    200ms blink
UART Baud:         115200
GPIO LED:          Pin 2


Building & Running
==================

1. Setup:
   export IDF_PATH=/path/to/esp-idf
   source $IDF_PATH/export.sh

2. Build:
   cd aoa_controller
   idf.py build

3. Flash:
   idf.py -p /dev/ttyUSB0 flash monitor

4. Test:
   python3 tools/send_sim_data.py --port /dev/ttyUSB0
   python3 tools/monitor.py --port /dev/ttyUSB0 --output log.csv


Performance
===========
